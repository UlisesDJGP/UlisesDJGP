### ESP32-S3 Wireless Image Classifier
**[Repository](https://github.com/JulissaVelazquez/IoT-image-classifier)** | **Role:** Core Developer

Distributed computer vision IoT system for real-time image classification using a client-server architecture.

**System Architecture:**
- ESP32-S3 edge device with OV2640 camera sensor and SSD1306 OLED display
- FastAPI backend server with TensorFlow/Keras inference engine
- MobileNetV2 CNN trained via Transfer Learning on Kaggle Dogs vs Cats dataset
- RESTful API communication through Ngrok tunnel

**Technical Implementation:**
- Client: Arduino C++ with ESP32 camera library and HTTP client
- Server: Python FastAPI with Uvicorn ASGI server
- Real-time image capture, wireless transmission, server-side inference, and edge feedback loop

**Tech Stack:** ESP32-S3, C++, Python, FastAPI, TensorFlow, Keras, OpenCV, Arduino
